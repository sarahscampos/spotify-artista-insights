# 🟢 Spotify Artista Insights Dashboard

![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)
![Spotify](https://img.shields.io/badge/Spotify-1ED760?style=for-the-badge&logo=Spotify&logoColor=white)
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=Python&logoColor=white)

Este é um dashboard interativo desenvolvido em Python que consome a API do Spotify para extrair e visualizar dados de qualquer artista da plataforma. O projeto foca na análise de discografia, métricas de duração de faixas e mapeamento de colaborações.

[Acesse o Dashboard](https://spotify-artista-insights.streamlit.app/)

---

## Funcionalidades

- **Busca Universal:** Introduza o ID de qualquer artista para carregar os seus dados instantaneamente.
- **Análise de Álbuns:** Seleção dinâmica de álbuns para visualizar a lista de faixas e capas.
- **Métricas de Performance (KPIs):** Visualização rápida da duração total do álbum, média de tempo por música e quantidade de parcerias (*feats*).
- **Gráficos Interativos:** Análise de variabilidade de tempo das faixas e proporção de colaborações utilizando Plotly.
- **Player Integrado:** Widget oficial do Spotify para ouvir as músicas diretamente no dashboard.

## Tecnologias Utilizadas

- [Streamlit](https://streamlit.io/): Framework para criação da interface web.
- [Spotipy](https://spotipy.readthedocs.io/): Biblioteca cliente para a API Web do Spotify.
- [Pandas](https://pandas.pydata.org/): Manipulação e tratamento dos dados em DataFrames.
- [Plotly Express](https://plotly.com/python/): Criação de gráficos dinâmicos e interativos.

## Pré-requisitos

- Python 3.14+
- Uma conta no [Spotify Developer Dashboard](https://developer.spotify.com/dashboard) para obter as suas credenciais.
